## 📖 About the Project

**Comprar** is a mobile application developed to help users manage their shopping and grocery lists with ease and efficiency. The application provides a seamless user interface where you can keep track of pending items to buy and check them off once purchased. 

This project was developed during the **React Native Journey by Rocketseat**, aiming to consolidate concepts of state management, componentization, and UI/UX design implementation in mobile development.

## ✨ Technical Features

- **Dynamic State Management:** Efficiently handles list states, ensuring the UI reflects real-time changes between `Pending` and `Purchased` items.
- **Item Registration:** Users can input and add new items to the shopping list dynamically.
- **Toggle Status:** Items can be easily toggled from "pending" to "purchased" (and vice versa), providing immediate visual feedback.
- **Item Deletion:** Ability to permanently remove items from the list, cleaning up the storage and view.
- **Empty State Handling:** Graceful UI fallbacks when no items are present in the list, improving the overall User Experience.
- **Custom UI Components:** Reusable and modularized components following the design system guidelines.

## 🚀 Getting Started

Follow the instructions below to set up the project locally on your machine.

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (LTS version)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- A package manager (npm, yarn, or pnpm)

### Installation

```bash
# Clone the repository
git clone [https://github.com/leonardomenezes7/comprar.git](https://github.com/leonardomenezes7/comprar.git)

# Navigate into the project directory
cd comprar

# Install the dependencies
npm install

# Start the application
npx expo start

```
